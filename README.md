# 825220075-Nicolas-Phi-KODE-PROGRAM
Dashboard Analitik Penerimaan Mahasiswa Baru (PMB) berbasis ASP.NET WebForms (VB.NET) + AdminLTE + Chart.js dengan star schema (SQL Server). Menyediakan KPI realisasi vs target, tren per tahun, komposisi gender, Top 10 sekolah/provinsi, serta filter rentang tahun dan fakultas. Repo ini memakai data contoh &amp; konfigurasi DB disamarkan.
